FAQ

The bot doesn't respond to my command. What should I do?

Try the following steps:

  1. Make sure Stashly is installed in your workspace.
  2. Invite the bot to the channel: type /invite @Stashly.
  3. Wait a moment and try again â€” the server may be starting up.
  4. If the issue persists, contact support.
Why does the Canvas get overwritten each time? (Free plan)

The Free plan overwrites the existing Canvas on each run to prevent unlimited Canvas accumulation. Upgrade to Pro to append new results to the same Canvas and build a running log.

The bot says "channel not found". How do I fix this?

Invite Stashly to the channel first: type /invite @Stashly in the channel you want to collect from.

Can I collect from private channels?

Yes, as long as the Stashly bot has been invited to that private channel.

I get a "missing scope" error after reinstalling.

Please uninstall the app completely from your workspace Settings â†’ Installed apps, then reinstall from scratch. This refreshes the OAuth scopes.
